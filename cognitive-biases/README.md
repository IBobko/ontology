# Cognitive Biases Catalog

This directory stores Markdown notes about cognitive biases and related interpretive frameworks.

## File index
- `summary.md` — overview of internal world models and common cognitive distortions.
- `thucydides-trap.md` — structured note on the Thucydides Trap.

## How to extend
When adding a new file to this catalog:
1. Create a new `.md` file in this directory.
2. Add one bullet to **File index** with:
   - filename;
   - a short one-line description.
3. Keep titles and filenames in English for consistency.
