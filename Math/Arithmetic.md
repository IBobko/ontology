# Арифметика

Арифметика — это раздел математики, который изучает числа, свойства чисел и базовые операции над ними.

## Основные арифметические операции

1. **Сложение** (`+`) — объединение двух чисел в одно значение.
   - Пример: `7 + 5 = 12`.
2. **Вычитание** (`-`) — нахождение разности между числами.
   - Пример: `10 - 4 = 6`.
3. **Умножение** (`×` или `*`) — многократное сложение.
   - Пример: `3 × 4 = 12`.
4. **Деление** (`÷` или `/`) — разбиение числа на равные части.
   - Пример: `12 / 3 = 4`.

## Порядок выполнения действий

При вычислении выражений используется следующий приоритет:

1. Действия в скобках.
2. Умножение и деление.
3. Сложение и вычитание.

Пример:

`2 + 3 × 4 = 2 + 12 = 14`

## Свойства операций

- **Переместительное свойство сложения:** `a + b = b + a`.
- **Переместительное свойство умножения:** `a × b = b × a`.
- **Сочетательное свойство сложения:** `(a + b) + c = a + (b + c)`.
- **Распределительное свойство:** `a × (b + c) = a × b + a × c`.

## Зачем нужна арифметика

Арифметика используется каждый день: при подсчёте времени, денег, покупок, расстояний и других практических задач.
