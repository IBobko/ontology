# The Main Principle of the System

## Основное наименование

**Принцип редукции различий** — основное название принципа.

## Более точные формулировки

Предпочтительные формулировки для точного описания механики:

- **Принцип редукции различий**.
- **Принцип управляемой редукции различий**.

Термин «редукция» указывает на сведение, сжатие и упрощение различий.

## Тезисная формулировка принципа

Любая система стремится свести множество различий к ограниченному, управляемому набору.
Она сохраняет только те различия, которые способна обрабатывать, а остальные либо устраняет, либо заменяет абстрактными категориями.

## Краткие ударные формулы

- **Система редуцирует различия до управляемого минимума.**
- **Система заменяет различия категориями.**
- **Система устраняет неуправляемые различия.**

## Статус принципа

Это верхнеуровневый принцип работы системы.

Для наибольшей концептуальной устойчивости приоритетной является формулировка через **редукцию**.
