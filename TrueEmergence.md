# Эмерджентность

Операция сознания по свёртке огромного количества взаимодействий в одну величину.

# Система

Множество микро-событий
↓
правило преобразования
↓
одна макро-величина


Эмерджентность — это не свойство реальности,
а правило преобразования множества элементов в одну форму описания.
