# README: папка `System`

Ниже перечислены файлы, которые находятся рядом с этим `README.md` в директории `System`, и кратко описано их содержание.

| Файл | Описание содержимого |
|---|---|
| `FormalSystem.md` | Что такое формальная система: алфавит, аксиомы, правила вывода, различие между синтаксисом и смыслом. |
| `PrincipleOfDifferenceFolding.md` | Принцип «свёртки различий»: как системы стремятся к устойчивости через снижение активных различий. |
| `Structure.md` | Различие между элементами, структурой и системой; объяснение, почему структура — это карта связей, а система — ещё и правила преобразования. |
| `SubjectInsideSystem.md` | Ограничения субъекта внутри системы: что он способен/не способен различать в рамках заданных правил. |
| `System.md` | Базовое определение системы: элементы, отношения, правила, границы применимости и типы систем. |
| `SystemAsForm.md` | Идея «системы как формы»: почему реактивность редко меняет форму без структурного различения. |
| `SystemRules.md` | Как правила системы внедряются и удерживаются: различие между принуждением и устойчивым воспроизводством правил. |
| `README.md` | Этот файл: навигация по содержимому директории `System`. |

## Поддиректории рядом

- `Protection/` — набор материалов о риторических механизмах самозащиты системы (подробная карта в `Protection/README.md`).
