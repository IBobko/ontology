# Защита системы

В этой папке собраны механизмы, с помощью которых система защищает себя от прямой ответственности.

## Список защит

1. [Размывание ответственности через обобщение](./ResponsibilityDilutionByGeneralization.md)  
   Суть: вопрос о конкретном субъекте переводится в общее правило, и персональная ответственность исчезает из обсуждения.

2. [Подмена ответственности объяснением](./ResponsibilitySubstitutionByExplanation.md)  
   Суть: вопрос «кто отвечает» подменяется объяснением «почему так произошло», что создаёт иллюзию ответа без признания ответственности.
