# Causality — конспект

## 1) Главная рамка
- Причинность снимается со статуса «внешнего закона мира».
- В опыте напрямую даны: события, их последовательность и фиксация сознанием.
- «Объективная причинность вне наблюдателя» — это гипотеза, а не непосредственный факт опыта.

## 2) Что важно не потерять
- Нельзя делать скачок от «нет внешнего закона» к «причинности нет вообще».
- Даже в минимальной схеме остаются:
  - повторяемость,
  - устойчивость переходов,
  - ожидание следующего состояния.
- Это уже структура, а не просто набор кадров.

## 3) Где рождается причинность
При многократном повторе маршрута `A → B → C`:
- после `A` возникает ожидание `B`,
- нарушение ожидания фиксируется как сбой,
- появляется различение «нормально / отклонение».

Итог:
- причинность = не «сила», а устойчивая связка различий,
- точнее: стабилизированный маршрут, воспринимаемый как связанный.

## 4) Механика перехода: «последовательность → связь»
1. **Поток различий**: есть только смена кадров.
2. **Фиксация**: `A` удерживается в памяти на фоне `B`.
3. **Наложение повторов**: множество случаев сжимается в схему `A → B`.
4. **Ожидание**: при новом `A` предактивируется `B`.
5. **Нарушение** (`A → X`): подтверждает, что связь уже была построена.

Рабочее определение:
- связь = устойчивое ожидание перехода, которое может подтверждаться или нарушаться.

## 5) Как из причинности делают «закон»
Различие между уровнями:
- ожидание: «скорее будет `B`»,
- обязательность: «должно быть `B`».

Подмена происходит так:
1. выделяют устойчивый маршрут,
2. повышают его вес,
3. подавляют альтернативы,
4. выносят результат как «внешний закон».

Цена фиксации:
- падение чувствительности к новому,
- игнор отклонений,
- защита схемы вместо работы со смыслом.

## 6) Интерпретация через SENSMAP
- `A`, `B` — узлы различений,
- `A → B` — маршрут,
- повторяемость — рост веса маршрута,
- ожидание — активация следующего узла до фактической фиксации.

Формула:
- причинность = маршрут с высоким весом + активированное предсказание.

## 7) Практический вывод: разфиксация без потери навигации
Различать 3 уровня:
1. поток различий,
2. маршруты (статистика переходов),
3. фиксация («должно быть только так»).

Разфиксировать нужно **только уровень 3**.

Правильная замена:
- было: `A → B` (обязательно),
- стало: `A → B` (высокая вероятность, но не единственный путь).

Эффект:
- сохраняется скорость (основной маршрут остаётся),
- допускаются альтернативы,
- отклонение читается как информация, а не как «ошибка мира».

## 8) Жёсткие итоговые формулировки
- Причинность не дана как внешний независимый закон.
- Она возникает из фиксации повторяющихся последовательностей и стабилизации маршрутов в смысловом поле.
- «Причина» — не скрытая сила, а закреплённое различие между состояниями.
- «Закон причинности» = маршрут + ожидание + запрет на отклонение.
- Разфиксация = снятие обязательности при сохранении веса маршрута и предсказательной функции.

## 9) Коротко в одну строку
Причинность в этой рамке — это не онтологическая «сила мира», а режим навигации сознания по устойчивым, но принципиально пересматриваемым маршрутам.
