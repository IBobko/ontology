# Рецептор

## Коротко

**Рецептор** — это не «любой элемент, который реагирует», а **специализированная структура**, которая переводит внешнее физическое воздействие в сигнал нервной системы.

## Что обязательно есть у рецептора

1. **Вход (стимул)**: свет, давление, температура, химический сигнал и т.д.
2. **Преобразование**: физическое/химическое воздействие → электрический (нейронный) сигнал.
3. **Передача дальше**: сигнал уходит в нервную систему для последующей обработки.

## Почему это не «контейнер реакции»

Слово «контейнер» слишком общее и не фиксирует функцию.

Рецептор делает больше, чем просто «реагирует»:

- фильтрует вход;
- чувствителен только к части диапазона;
- подавляет одни различия и усиливает другие;
- формирует первичную различимость для системы.

То есть рецептор — это **входной преобразователь + фильтр**, а не просто пассивный ответчик.

## Рабочая формулировка

> Рецептор — это узел, где внешний поток воздействий преобразуется в различимый сигнал системы.

## Пример: температурная чувствительность

Нет одного «рецептора всего тепла». Есть разные каналы/рецепторы с разными диапазонами активации:

- одни активны в умеренном диапазоне;
- другие — в более тёплом;
- третьи — при потенциально повреждающем перегреве.

Следствие: уже на уровне рецепторов реальность разбивается на рабочие диапазоны.

## Связь с темой различения

Рецептор — это первый уровень, где:

- поток данных уже **сжимается**;
- различия уже **отбираются**;
- часть полноты исходного воздействия **теряется**.

Иными словами, до «осознанной интерпретации» есть слой предварительного ограничения.
